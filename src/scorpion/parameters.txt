REST API Documentation 

rest-api-name:EV
rest-api-brief:REST API EV provides methods to support basic easy value request (get/set of parameter values)
rest-api-method-request:SUMMARY
rest-api-method-brief:Returns a simplified summary of the API.
rest-api-method-syntax:v.api/apis/EV/SUMMARY
rest-api-method-response:JSON data containing supported methods or error {"error":"error-message-text"}
rest-api-method-request:GET
rest-api-method-brief:Retrieves a value for a simple parameter.
rest-api-method-syntax:v.api/apis/EV/GET/parameter?varid=<VARID> <br>v.api/apis/EV/GET/parameter/<VARID>
rest-api-method-response:JSON data containing <VARID> and value or error {"error":"error-message-text"}
rest-api-method-request:GET
rest-api-method-brief:Retrieves value(s) for one or more parameters .
rest-api-method-syntax:v.api/apis/EV/GET/parameters?varid=<VARID1>,<VARID2>... <br>v.api/apis/EV/GET/parameters/<VARID1>,<VARID2>...
rest-api-method-response:JSON data containing a list of <VARID>s and values or error {"error":"error-message-text"}
rest-api-method-request:SET
rest-api-method-brief:Assigns a value to a simple parameter. JSON parameters are restricted.
rest-api-method-syntax:v.api/apis/EV/SET/parameter?<VARID>=<VALUE> <br>v.api/apis/EV/SET/parameter/<VARID>/<VALUE>
rest-api-method-response:JSON data containing <VARID> and execition statue or common error {"error":"error-message-text"}
rest-api-method-request:SET
rest-api-method-brief:Assigns value(s) to one or more parameters. JSON parameters are restricted.
rest-api-method-syntax:v.api/apis/EV/SET/parameters?<VARID1>=<VALUE1>&<VARID2>=<VALUE2>... <br>v.api/apis/EV/SET/parameters/<VARID1>,<VARID2>.../<VALUE1>,<VALUE2>...
rest-api-method-response:JSON data containing list of <VARID>s and execution status or common error {"error":"error-message-text"}

rest-api-method-brief:Returns a simplified summary of the API.
rest-api-method-syntax:v.api/apis/PT/SUMMARY
rest-api-method-response:JSON format data containing supported methods or error {"error":"error-message-text"}
rest-api-method-request:CALL
rest-api-method-brief:Performs a direct passthrough call of the json-rpc call to the device.
rest-api-method-syntax:v.api/apis/PT/CALL/jsonrpc/<base64 rpc json>
rest-api-method-response:JSON format data of RPC or error {"error":"error-message-text"}

rest-api-name:RT
rest-api-brief:REST API RT provides a method to allow routes to be made for applicable devices.
rest-api-method-request:SUMMARY
rest-api-method-brief:Returns a simplified summary of the RT API.
rest-api-method-syntax:v.api/apis/RT/SUMMARY
rest-api-method-response:JSON data containing supported methods or error {"error":"error-message-text"}
rest-api-method-request:GET
rest-api-method-brief:Retrieves Router output info
rest-api-method-syntax:v.api/apis/RT/GET/router-destination/<ROUTER-VARID>/<OUTPUT-ID>
rest-api-method-response:JSON data containing input vs output info,  or error {"error":"error-message-text"}
rest-api-method-request:ROUTE
rest-api-method-brief:Route an input to output, optionally specify aspen ID/aspen label or short label or state-type abbreviation
rest-api-method-syntax:v.api/apis/RT/ROUTE/<ROUTER-VARID>/<OUTPUT-ID>/<INPUT-ID>[/aspen/<ASPEN-ID>][/state-type/<STATE-TYPE-ABBRS>]
rest-api-method-response:JSON data containing ROUTE status,  or error {"error":"error-message-text"}
rest-api-method-request:UNROUTE
rest-api-method-brief:Un-Route an input from output or all inputs if no input is provided
rest-api-method-syntax:v.api/apis/RT/UNROUTE/<ROUTER-VARID>/<OUTPUT-ID>[/<INPUT-ID>]
rest-api-method-response:JSON data containing UNROUTE status,  or error {"error":"error-message-text"}
